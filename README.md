# E-Commerce Sales Trend & Product Performance Analysis

## Project Overview
This project analyzes e-commerce transaction data to identify sales trends, top-performing products, customer repeat behavior, and category performance.

## Objectives
- Understand monthly sales trends
- Identify top-selling products
- Analyze repeat customer rate
- Evaluate category contribution

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Top products generated significant revenue contribution
- Seasonal monthly fluctuations identified
- Repeat customers showed higher long-term value
- Certain categories dominated sales share

## Files
- `Ecommerce_Sales_Analysis.ipynb`
- `ecommerce_sales.csv`
